## Project setup

0. Enable pnpm
```
$ corepack enable pnpm
```

1. Install deps
```
$ pnpm install
```

## Project commands

- Run app
```
$ pnpm run dev
```

- Format
```
$ pnpm run format
```

- Lint
```
$ pnpm run lint
```

- Deploy to cloudflare workers
```
$ pnpm run deploy
```
